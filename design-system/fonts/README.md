# Fonts

Both fonts are loaded from Google Fonts in `colors_and_type.css`:

- **Press Start 2P** — https://fonts.google.com/specimen/Press+Start+2P
- **JetBrains Mono** — https://fonts.google.com/specimen/JetBrains+Mono

No local font files are needed. If you want to self-host, download the `.woff2`
files from Google Fonts into this folder and replace the `@import` at the top
of `colors_and_type.css` with `@font-face` blocks pointing here.
